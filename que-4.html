what are the different types of list in html ? with example

There are three types of lists in HTML: unordered lists, ordered lists, and description lists. 1. Unordered List: An unordered list is created using the `

    ` tag. Each list item is represented by the `
  • ` tag. By default, the list items are marked with bullets. Here's an example: ```html
    • Item
    • Item
    • Item
    • Item
    ``` 2. Ordered List: An ordered list is created using the `
      ` tag. Similar to unordered lists, each list item is represented by the `
    1. ` tag. By default, the list items are numbered. Here's an example: ```html
      1. First item
      2. Second item
      3. Third item
      4. Fourth item
      ``` 3. Description List: A description list is created using the `
      ` tag. It consists of terms and their corresponding descriptions. The terms are represented by the `
      ` tag, and the descriptions are represented by the `
      ` tag. Here's an example: ```html
      Coffee
      - black hot drink
      Milk
      - white cold drink
      ```
