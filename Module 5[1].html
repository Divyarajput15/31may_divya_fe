Question 1.   What are the new tags added in HTML5?

New added tags

  • 1] 'article'= Represents an independent piece of content of a document , like a blog entry or newspaper article.
  • 2] 'aside'= defines content aside from the page content.
  • 3] 'audio'= defines an audio file.
  • 4] 'canvas'= this is used for dynamic graphics on the fly,such as graphs or games.
  • 5] 'command'= Represents a command the user can invoke.
  • 6] 'datalist'= together with the a new list attribute for input can be used to make comboboxes.
  • 7] 'details'= Represents additional information or controls which the user can obtain on demand.
  • 8] 'embed'= defines external interactive content or plugin.
  • 9] 'figure'= Represents that self-contained flow content ,typically referenced as a single unit from the main flow of the document.
  • 10] 'footer'= Represents a footer for a section and can contain information about the author,copyright,information,etc.
  • 11] 'header'= Represents a group of introductory or navigational aids.
  • 12] 'hgroup'= Represents the header of a section.
  • 13] 'keygen'= Represents controls for key pair generation.
  • 14] 'mark'= defines marked/highlighted text.
  • 15] 'meter'= Represents a measurement,such as disk usage.
  • 16] 'nav'= defines navigation links
  • 17] 'output'= Represents some type of output,such as from a calculation done through scripting.
  • 18] 'progress'= Represents a completion of a task , such as downloading or when performing a series of expensive operations.
  • 19] 'section'= Represents a document or application section.
  • 20] 'time'= Represents a date and/or time.
  • 21] 'video'= defines a video file.
  • 22] 'wbr'= Represents a line break opportunity.
