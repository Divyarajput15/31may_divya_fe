1. What are the benefits of using CSS?

- Separation of Style and Content: CSS allows for a clear distinction between the structure (HTML) and the presentation (CSS) of a webpage, making maintenance and updates easier.
- Consistency: CSS enables a uniform look across multiple pages by centralizing styling rules.
- Efficiency: Code becomes cleaner as styles are defined in one place rather than repeated inline.
- Flexibility: Design elements like fonts, colors, and layout can be customized with precision.

2. What are the disadvantages of CSS?

- Browser Compatibility: CSS can behave differently in various browsers, requiring additional effort for cross-browser consistency.
- Learning Curve: Understanding CSS thoroughly, especially its advanced features, demands a learning curve.
- Design Limitations: Complex designs might be challenging to achieve due to CSS limitations.
- Maintenance Complexity: Large CSS files can become unwieldy, making maintenance and updates challenging.

3. What is the difference between CSS2 and CSS3?

- CSS2: Focused on basic styling features like text formatting, positioning, and layout.
- CSS3: Introduces advanced capabilities including animations, transitions, gradients, and rounded corners.

4. Name a few CSS style components.

- Fonts: CSS allows control over font properties such as font-family, font-size, and font-weight.
- Colors: Styling attributes like color, background-color, and border-color define color choices.
- Margins/Padding: CSS properties like margin and padding adjust spacing around elements.

5. What do you understand by CSS opacity?

- CSS Opacity: Opacity defines the transparency level of an element, ranging from 0 (fully transparent) to 1 (fully opaque).
- Example: To apply a semi-transparent red background:
.transparent-box {
background-color: rgba(255, 0, 0, 0.5);
}

6. How can the background color of an element be changed?

- Changing Background Color: The background-color property sets the background color of an element.
- Example: To create a green background:
.colored-box {
background-color: #00ff00;
}

7. How can image repetition of the background be controlled?

- Controlling Background Repetition: The background-repeat property manages how a background image repeats.
- Example: To repeat an image only horizontally:
.patterned-bg {
background-repeat: repeat-x;
}

8. What is the use of the background-position property?

- Background Positioning: background-position places a background image within its container.
- Example: To center an image horizontally and align it to the top:
.bg-position {
background-position: center top;
}

9. Which property controls the image scroll in the background?

- Background Attachment: background-attachment determines if a background image scrolls with content.
- Example: To fix an image while scrolling:
.fixed-bg {
background-attachment: fixed;
}

10. Why should background and color be used as separate properties?

- Separation for Clearer Styling: Isolating background and color properties improves readability and prevents conflicts.
- Example: To set background and text color:
.styled-element {
background-color: #f0f0f0;
color: #333;
}

11. How to center block elements using CSS1?

- Centering with CSS1: Set a fixed width and use auto margins for horizontal centering.
- Example: To center a block with 50% width:
.centered {
width: 50%;
margin: 0 auto;
}

12. How to maintain the CSS specifications?

- Staying Updated: Regularly consult W3C's CSS specifications and adhere to recommended practices.

13. What are the ways to integrate CSS as a web page?

- Inline Styles: Apply styles directly within HTML tags using the style attribute.
- Embedded Styles: Insert CSS within "" tags in the HTML . - External Stylesheets: Link to an external CSS file using the tag in the HTML .

14. What is embedded style sheets?

Embedded Styles CSS rules placed within applying specifically to that document.

15. What are external style sheets?

- External Stylesheets: Separate CSS files linked to HTML documents via the element, promoting reusability.

16. Advantages and disadvantages of using external style sheets?

- Advantages: Consistent design, caching for faster load times, and centralized maintenance.
- Disadvantages: Additional HTTP request, potential for specificity conflicts.

17. What is the meaning of the CSS selector?

- CSS Selector: A CSS selector is a pattern that identifies which HTML elements the associated styles should be applied to.

18. What are the media types allowed by CSS?

- Media Types: CSS supports various media types such as screen, print, and speech, allowing styles to be adapted for different outputs.

19. What is the rule set?

- CSS Rule Set: A rule set consists of a selector and a declaration block. The selector specifies the target element, and the declaration block contains property-value pairs for styling.
